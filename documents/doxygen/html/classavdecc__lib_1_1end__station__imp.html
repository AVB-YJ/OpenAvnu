Deduce what else needs to be read from the rx'd frame.

This function looks inside rx'd descriptors and deducees which descriptors need to be read in the background next. There are two lists that are maintained for reading descriptors. The m_backbround_read_pending list where descriptors are queued before being sent and the m_backbround_read_inflight list the contains read requests that are on "the wire".

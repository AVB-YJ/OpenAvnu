virtual AVDECC_CONTROLLER_LIB32_API uint8_t* STDCALL locale_identifier ( )
pure virtual

The identifier is a UTF-8 string that contains one to three components such as a language code, a region code, or a variant code, separated by the dash character. Examples of valid locale identifiers are en-US for English in the US, en-AU for English in Australia, haw-US for Hawaiian in the US, and fr-CA for French in Canada.

virtual AVDECC_CONTROLLER_LIB32_API uint8_t* STDCALL object_name ( )
pure virtualinherited
반환값
The name of the descriptor object. This may be user set through the use of a SET_NAME command. The object name should be left blank (all zeros) by the manufacturer, with the manufacturer defined value being provided in a localized form via the localized descripton field. By leaving this field blank an AVDECC Controller can determine if the user has overridden the name and can use this name rather than the localized name.

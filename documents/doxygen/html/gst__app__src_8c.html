send_data_status() - Send the status to transmitter : 0/1 for stop and start respectively

This function sends 0 to transmitter to stop transmitting the data for a while, there is enough of data available with gstreamer to handle. sends 1 to transmitter to start back again with the data transmission.

Returns 0

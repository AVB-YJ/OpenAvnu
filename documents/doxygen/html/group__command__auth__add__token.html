매개변수
pconst pointer to command_auth_add_token structure to read from.
basepointer to raw memory buffer to write to.
posoffset from base to write the field to;
lenlength of the raw memory buffer;
반환값
-1 if the buffer length is insufficent, otherwise the offset of the octet following the structure in the buffer.

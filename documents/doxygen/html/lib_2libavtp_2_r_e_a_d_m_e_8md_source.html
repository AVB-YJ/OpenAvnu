1 # About
2 
3 Open source implementation of Audio Video Transport Protocol (AVTP) specified
4 in IEEE 1722-2016 spec.
5 
6 Libavtp is under BSD License. For more information see LICENSE file.
7 
8 # Build
9 
10 Before building libavtp make sure you have all the required software installed
11 in your system. Below are the requirements and their tested versions:
12 
13 * Meson >= 0.43
14 * Ninja >= 1.8.2
15 
16 The first step to build libavtp is to generate the build system files.
17 
18 ```
19 $ meson build
20 ```
21 
22 Then build libavtp by running the following command. The building artifacts
23 will be created under the build/ in the top-level directory.
24 
25 ```
26 $ ninja -C build
27 ```
28 
29 To install libavtp on your system run:
30 ```
31 $ sudo ninja -C build install
32 ```
33 
34 # AVTP Formats Support
35 
36 AVTP protocol defines several AVTPDU type formats (see Table 6 from IEEE
37 1722-2016 spec). Libavtp doesn't support all of them yet. The list of supported
38 formarts is:
39 * AAF (PCM encapsulation only)
40 
41 # Examples
42 
43 The `examples/` directory in the top-level directory provides example
44 applications which demonstrate the libavtp functionalities. To build an
45 example application run `$ ninja -C build <example name>`.
46 
47 Information about what exactly each example application does and how it works
48 is provided in the beginning of the .c file from each application.

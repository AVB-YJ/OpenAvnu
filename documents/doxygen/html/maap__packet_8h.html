상세한 설명

Support for packing and unpacking MAAP packets.

These functions convert a binary stream of bytes into a MAAP_Packet structure, or vice versa.

int compare_mac_addresses ( uint64_t  local_mac,
uint64_t  remote_mac 
)

Compare two MAC addresses, starting with the least-significant bytes, to determine if the local address is numerically lower.

주의
This is equivalent to the compare_MAC function in IEEE 1722-2016 B3.6.4
매개변수
local_macThe MAC Address for the local network interface of the computer running this application
remote_macThe source MAC Address in a MAAP packet that was received
반환값
1 if the local_mac is less than then remote_mac, 0 otherwise

uint64_t convert_mac_address ( const uint8_t  macaddr[])

Convert a byte-order MAC Address into a 64-bit number.

The 64-bit number format is used to fill MAAP_Packet::DA and MAAP_Packet::SA in the MAAP_Packet structure.

매개변수
macaddrPointer to the byte-order MAC Address to convert
반환값
A 64-bit number equivalent to the supplied MAC Address
